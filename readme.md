# Waka-Anki

二次元风格的可视化间隔重复记忆卡片应用。支持快速制卡、SM-2 算法复习、数据统计，纯前端 + Node.js 本地服务，无需安装数据库。

## 快速开始

```bash
# 方式一：双击运行
start.bat

# 方式二：命令行
node server.js
# 浏览器打开 http://localhost:3456
```

纯前端模式：直接用浏览器打开 `anki.html`，数据存储在 localStorage。

## 功能

- **制卡**：单张/批量创建卡片，支持图片粘贴与上传，自动检测分隔符
- **管理**：卡组、文件夹层级组织，拖拽移动，右键菜单操作，卡片搜索编辑
- **复习**：SM-2 间隔重复算法，学习/复习/重学状态调度，按键快捷键（空格翻面，1-4评分）
- **统计**：复习热力图（7/30/180天）、卡片数量与间隔概览
- **Markdown 渲染**：卡片内容支持完整 Markdown 语法渲染（标题、粗体、斜体、删除线、内联代码、代码块、列表、引用、链接、图片、表格、分割线）
- **导入导出**：Markdown 格式（Q:/A:/GROUP:）导出为 ZIP，支持重新导入；兼容 Anki CSV/TSV

## 项目结构

```
anki.html          # 主应用（单页面）
server.js          # Node.js 本地服务（端口 3456）
js/
  db.js            # 数据层（localStorage）
  scheduler.js     # SM-2 间隔重复算法
data/              # JSON 持久化数据
source/            # 卡片生成脚本与源文件
```

## 技术栈

HTML5 + CSS3 + 原生 JavaScript，Tailwind CSS（CDN），JSZip（CDN）。零前端依赖，Node.js 服务仅使用内置模块。

## 数据格式

导出的 Markdown 格式：

```markdown
# 卡片ID
Q: 正面内容
A: 背面内容
GROUP: 卡组名
```

兼容 Anki CSV 导入（UTF-8，逗号分隔，字段：正面/背面/标签/卡组）。

## Markdown 语法支持

卡片正面和背面内容均支持以下 Markdown 语法，导入和手动添加时自动转换为 HTML 渲染：

| 语法 | 示例 | 效果 |
|------|------|------|
| 粗体 | `**粗体**` 或 `__粗体__` | **粗体** |
| 斜体 | `*斜体*` 或 `_斜体_` | *斜体* |
| 粗斜体 | `***粗斜体***` | ***粗斜体*** |
| 删除线 | `~~删除线~~` | ~~删除线~~ |
| 内联代码 | `` `code` `` | `code` |
| 代码块 | ```` ``` ```` 包裹 | 语法高亮代码块 |
| 标题 | `# H1` `## H2` `### H3` | 六级标题 |
| 无序列表 | `- 项目` 或 `* 项目` | 无序列表 |
| 有序列表 | `1. 项目` | 有序列表 |
| 引用 | `> 引用` | 引用块 |
| 链接 | `[文本](url)` | 可点击链接 |
| 图片 | `![alt](url)` | 图片显示 |
| 分割线 | `---` `***` `___` | 水平分割线 |
| 表格 | `\| A \| B \|` 等 | HTML 表格 |
