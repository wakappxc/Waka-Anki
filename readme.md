# Waka-Anki

二次元风格的可视化间隔重复记忆卡片应用。支持快速制卡、SM-2 算法复习、数据统计。纯前端 PWA，可部署到 GitHub Pages，无需服务器。

## 快速开始

```bash
# 开发模式（Node.js 本地服务）
node server.js
# 浏览器打开 http://localhost:3456
```

纯前端模式：直接用浏览器打开 `anki.html`，所有数据存储在 localStorage，支持 PWA 离线使用。

## 功能

- **制卡**：单张/批量创建卡片，支持图片粘贴插入，自动检测分隔符
- **管理**：卡组、文件夹层级组织，右键菜单操作（重命名、移动、清空、删除），卡片搜索编辑
- **复习**：SM-2 间隔重复算法，学习/复习/重学状态调度，按键快捷键（空格翻面，1-4 评分），卡片自适应内容高度
- **统计**：复习热力图（7/30/180 天）、卡片数量与平均间隔概览
- **Markdown 渲染**：卡片内容支持完整 Markdown 语法渲染（标题、粗体、斜体、删除线、内联代码、代码块、列表、引用、链接、图片、表格、分割线）
- **导入**：Markdown 格式（Q:/A:/GROUP:）导入，自动检测本地图片引用并支持匹配本地文件转换为内嵌图片
- **导出**：卡片按卡组导出为 ZIP（含 MD 文件 + 附件图片）；统计页支持导出全部数据为 JSON 备份（卡片、复习记录、热力图数据），支持从备份 JSON 恢复

## 项目结构

```
anki.html          # 主应用（单页面 PWA）
server.js          # Node.js 本地服务（端口 3456，开发用）
sw.js              # Service Worker（离线缓存）
manifest.json      # PWA 清单
js/
  db.js            # 数据层（localStorage，三个 key：waka_ankiweb / waka_revlog / waka_timehot）
  scheduler.js     # SM-2 间隔重复算法
data/              # JSON 持久化数据（开发模式服务端使用）
source/            # 卡片生成脚本与源文件
```

## 技术栈

HTML5 + CSS3 + 原生 JavaScript，Tailwind CSS（CDN），JSZip（CDN）。零前端依赖，PWA 离线可用，支持 GitHub Pages 部署。

## 数据格式

### Markdown 导出格式

```markdown
# 卡片ID
Q: 正面内容
A: 背面内容
GROUP: 卡组名
```

图片以 data URI 嵌入，或提取为 `附件/` 目录下的文件。

### JSON 备份格式（统计页导出）

```json
{
  "version": 1,
  "exportedAt": "2026-05-17T...",
  "ankiweb": { "decks": [...], "notes": [...], "cards": [...], "folders": [...] },
  "revlog": { "revlog": [...] },
  "timehot": { "days": { "2026-05-17": 5 } }
}
```

可在统计页通过「导入数据」按钮恢复。

## Markdown 语法支持

卡片正面和背面内容均支持以下 Markdown 语法，导入和手动添加时自动转换为 HTML 渲染：

| 语法 | 示例 | 说明 |
|------|------|------|
| 粗体 | `**粗体**` 或 `__粗体__` | |
| 斜体 | `*斜体*` 或 `_斜体_` | |
| 粗斜体 | `***粗斜体***` | |
| 删除线 | `~~删除线~~` | |
| 内联代码 | `` `code` `` | |
| 代码块 | ```` ``` ```` 包裹 | 可选语言标识 |
| 标题 | `# H1` ~ `###### H6` | 六级标题 |
| 无序列表 | `- 项目` 或 `* 项目` | |
| 有序列表 | `1. 项目` | |
| 引用 | `> 引用` | 支持嵌套 |
| 链接 | `[文本](url)` | 新窗口打开 |
| 图片 | `![alt](url)` | 支持 data URI 和本地文件 |
| 分割线 | `---` `***` `___` | |
| 表格 | `\| A \| B \|` | 支持对齐（`:---` `:---:` `---:`） |
